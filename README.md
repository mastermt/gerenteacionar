# Gerente Acionar


Remote administration tool (Like BO2K).

It comes with a client and a server (you can use in place of explorer.exe).
For use in School Lab or Office Network. Administração Remota.

This project started in 1996 with an MVP in Visual Basic 6, it was rebuilt in Delphi 3. Currently using Delphi Sidney Community Edition.

It is even in use in 3 schools.


Administração Remota: Este programa está em uso em duas unidades (escolas, em laboratórios de línguas - onde cada aluno usa um micro para aprender inglês ou espanhol).

O ACIONAR é colocado no lugar do windows explorer e possui botões para acesso a cada programa.

O GERENTE fica em uma estação de controle onde pode-se:

    Iniciar um programa nas estações;
    Listar/Matar processos;
    Mapear unidade de rede;
    Mudar Resolução da Tela;
    Mudar Papel de Parede;
    Bloquear o uso de programas;
    Verificar/Monitorar a URL das estações;
    ETC;
